1.ACWing 795.前缀和

输入一个长度为 n的整数序列。
接下来再输入 m个询问，每个询问输入一对 l,r。
对于每个询问，输出原序列中从第 l个数到第 r个数的和。
输入格式
第一行包含两个整数 n和 m。
第二行包含 n个整数，表示整数数列。
接下来 m行，每行包含两个整数 l和 r，表示一个询问的区间范围。
输出格式
共 m行，每行输出一个询问的结果。
数据范围
1≤l≤r≤n,
1≤n,m≤100000,−1000≤数列中元素的值≤1000
输入样例：
5 3
2 1 3 6 4
1 2
1 3
2 4
输出样例：
3
6
10

解题思路：

D:\MyBlog\source_posts\images\ACWing_795.png

若对b分析可以得出，b[r]-b[l-1]即为l到r上的a[i]的和。

解题代码：

#include<iostream>
#include<algorithm>
using namespace std;
int n,m;
int l,r;
int arr[100010];
int b[100010];//
int main()
{
    scanf("%d%d",&n,&m);
    for(int i=1;i<=n;i++)
    {
        scanf("%d",&arr[i]);
        b[i]=b[i-1]+arr[i];
    }
    for(int i=1;i<=m;i++)
    {
        scanf("%d%d",&l,&r);
        printf("%d\n",b[r]-b[l-1]);
    }
    return 0;
}

2.ACWing 796.子矩阵的和

输入一个 n行 m列的整数矩阵，再输入 q个询问，每个询问包含四个整数 x1,y1,x2,y2，
表示一个子矩阵的左上角坐标和右下角坐标。
对于每个询问输出子矩阵中所有数的和。
输入格式
第一行包含三个整数 n，m，q。
接下来 n行，每行包含 m个整数，表示整数矩阵。
接下来 q行，每行包含四个整数 x1,y1,x2,y2，表示一组询问。
输出格式
共 q行，每行输出一个询问的结果。
数据范围
1≤n,m≤1000,
1≤q≤200000,
1≤x1≤x2≤n,
1≤y1≤y2≤m,
−1000≤矩阵内元素的值≤1000
输入样例：
3 4 3
1 7 2 4
3 6 2 8
2 1 2 3
1 1 2 2
2 1 3 4
1 3 3 4
输出样例：
17
27

解题思路：
思路和第一题差不多，只需要将a和b改成二维数组，对每一行的求前缀和即可

解题代码：

#include<iostream>
#include<cstdio>
using namespace std;
int n,m,q;
int arr[1010][1010];
int b[1010][1010];
int x1,x2,y1,y2;
int sum=0;
int main()
{
    scanf("%d%d%d",&n,&m,&q);
    for(int i=1;i<=n;i++)
    {
        for(int j=1;j<=m;j++)
        {
            scanf("%d",&arr[i][j]);
            b[i][j]=arr[i][j]+b[i][j-1];
        }
    }
    for(int i=0;i<q;i++)
    {
        sum=0;
        scanf("%d%d%d%d",&x1,&y1,&x2,&y2);
        for(int j=x1;j<=x2;j++)
        {
            sum+=(b[j][y2]-b[j][y1-1]);
        }
        printf("%d\n",sum);
    }
    return 0;
}

3.ACWing 797.差分

输入一个长度为 n的整数序列。
接下来输入 m个操作，每个操作包含三个整数 l,r,c，表示将序列中 [l,r]之间的每个数加上 c。
请你输出进行完所有操作后的序列。
输入格式
第一行包含两个整数 n和 m。
第二行包含 n个整数，表示整数序列。
接下来 m行，每行包含三个整数 l，r，c，表示一个操作。
输出格式
共一行，包含 n个整数，表示最终序列。
数据范围
1≤n,m≤100000,
1≤l≤r≤n,
−1000≤c≤1000,
−1000≤整数序列中元素的值≤1000
输入样例：
6 3
1 2 2 1 2 1
1 3 1
3 5 1
1 6 1
输出样例：
3 4 5 3 4 2

解题思路：

D:\MyBlog\source_posts\images\差分.png

什么意思呢，就是用前缀和数组的话其实在区间内加上一个数和减去一个数对b数组来说是不变的，唯一会变的只有左右边界的值，所以只需要对左右边界进行+c(c为区间内操作的数字)和-c即可，然后再对b数组进行前缀和即可。

解题代码：

#include <iostream>
using namespace std;
int l,r,c;
long long int n,m,sum;
long long int arr[100010];
long long int b[100010];
void insert(int l,int r,int c)
{
    b[l]+=c;
    b[r+1]-=c;
}
int main()
{
    scanf("%ld%ld",&n,&m);
    for(int i=1;i<=n;i++)
    {
        scanf("%ld",&arr[i]);
        insert(i,i,arr[i]);
    }
    for(int i=1;i<=m;i++)
    {
        scanf("%d%d%d",&l,&r,&c);
        insert(l,r,c);
    }
    for(int i=1;i<=n;i++)
    {
        b[i]+=b[i-1];
    }
    for(int i=1;i<=n;i++)
    {
        printf("%ld ",b[i]);
    }
}
