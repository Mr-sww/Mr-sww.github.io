1.ACWing 789.数的范围

给定一个按照升序排列的长度为 n的整数数组，以及 q个查询。
对于每个查询，返回一个元素 k的起始位置和终止位置（位置从 0开始计数）。
如果数组中不存在该元素，则返回 -1 -1。
输入格式
第一行包含整数 n和 q，表示数组长度和询问个数。
第二行包含 n个整数（均在 1∼10000范围内），表示完整数组。
接下来 q行，每行包含一个整数 k，表示一个询问元素。
输出格式共 q行，每行包含两个整数，表示所求元素的起始位置和终止位置。
如果数组中不存在该元素，则返回 -1 -1。
数据范围
1≤n≤100000
1≤q≤10000
1≤k≤10000
输入样例：
6 3
1 2 2 3 3 4
3
4
5
输出样例：
3 4
5 5
-1 -1

解题思路：

只需要将查询元素与数组中间元素进行比较，
case 1: 查询元素等于中间元素，则直接返回下标。
case 2: 查询元素大于中间元素，则递归调用函数，查找右半边。
case 3: 查询元素小于中间元素，则递归调用函数，查找左半边。
然后向前和向后寻找，即可得到起始位置和终止位置。
解题代码：

#include<iostream>
#include<algorithm>
using namespace std;
int n,q;
int k;
int st,e;
int arr[100010];
void found(int l,int r)
{
    int mid=(l+r)>>1;
    if(l>r)
    {
        cout<<"-1 -1"<<endl;
    }
    else
    {
        if(arr[mid]==k)
        {
            for(int i=l;i<=mid;i++)
            {
                if(arr[i]==k)
                {
                    st=i;
                    break;
                }
            }
            for(int i=r;i>=mid;i--)
            {
                if(arr[i]==k)
                {
                    e=i;
                    break;
                }
            }
            cout<<st<<" "<<e<<endl;
            return ;
        }
        else
        {
            if(arr[mid]>k) found(l,mid-1);
            else found(mid+1,r);
        }
    }
}
int main()
{
    cin>>n>>q;
    for(int i=0;i<n;i++)
    {
        cin>>arr[i];
    }
    for(int i=0;i<q;i++)
    {
        cin>>k;
        found(0,n-1);
    }
    return 0;
}

2.ACWing 790.数的三次方根

给定一个浮点数 n，求它的三次方根。
输入格式
共一行，包含一个浮点数 n。
输出格式
共一行，包含一个浮点数，表示问题的解。
注意，结果保留 6位小数。
数据范围
−10000≤n≤10000
输入样例：
1000.00
输出样例：
10.000000

解题思路：
思路与第一题一样，区别在于一开始的上下限制并非0和n而是为了找到所有情况选择-10000和10000。分别找到中间值的三次方根，判断与查询值大小，然后递归调用函数，找到所有情况，然后取最小值。

解题代码：

#include<iostream>
#include<cmath>
#include<cstdio>
using namespace std;
double n;
void found(double l,double r)
{
    double mid=(l+r)/2;
    if(r-l<=1e-7)
    {
        printf("%.6lf\n",l);
        return ;
    } 
    if(pow(mid,3)>=n) found(l,mid);
    else found(mid,r);
}
int main()
{
    scanf("%lf",&n);
    found(-10000,10000);
    return 0;
}
